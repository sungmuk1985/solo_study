머피의 법칙 (Murphy’s Law)
 - 될 수 있는 건 다 최악의 타이밍에 일어난다.

파킨슨의 법칙 (Parkinson’s Law)
 - 일은 주어진 시간만큼 질질 늘어난다.

파레토의 법칙 (Pareto Principle, 80/20 법칙)
 - 결과의 80%는 20%의 원인에서 나온다.

호프스태터의 법칙 (Hofstadter’s Law)
 - 생각보다 오래 걸리고, 그걸 알고 있어도 더 오래 걸린다.
   
피터의 원리 (Peter Principle)
 - 사람은 결국 자기 능력보다 높은 자리까지 승진해서 무능해진다.
